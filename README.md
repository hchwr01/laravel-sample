# Laravel Sample (/laravel)
Laravelのサンプルアプリケーションです。

---
## ToDoリスト (/laravel/todos)
Laravel (PHP/MySQL)を用いて、以下の要件を満たす ToDo リストシステムの構築を実施。

### 概要
- やるべき作業（以下 ToDo）をリストとして管理し、閲覧・登録・更新・削除・完了管理ができる Web アプリケーションを作成。
- ブラウザから操作できる Web ページとして実装。

### 前提条件
- 単一ユーザー利用 を前提。
- 認証・ログイン機能の実装は不要。
- 安全性・保守性を意識した実装。

### 機能要件
- ブラウザから URL を入力することで ToDo リストの Web ページにアクセスできる。
- ページ表示時に、過去に登録された ToDo の一覧が表示される。
- 新しい ToDo を登録できる。
- 登録済み ToDo の内容を修正できる。
- 登録済み ToDo を削除できる。
- 登録済み ToDo を完了状態に変更できる。
- 上述の機能にページ遷移を伴わない、シングルページアプリケーション(SPA)であること。

### 技術要件
- 言語：PHP
- データベース：MySQL
- フレームワーク：Laravel
- ToDo データは MySQL で管理すること

### SPA要件
- ページ遷移を行わず、ページリロード無しで CRUD 操作が完結すること。
- Ajax / fetch / axios 等の利用。
- フロントエンドフレームワーク（Vue / React 等）の使用は 任意。
- SPA の実装品質やフロントエンドの作り込みは 必須条件ではない。
